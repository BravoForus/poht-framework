# Proof-of-Holonic-Trust (PoHT)

**The canonical, immutable trust primitive for autonomous systems, AI governance, and institutional oversight.**

[![Zenodo DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.19033056.svg)](https://doi.org/10.5281/zenodo.19033056)

### Core Stack (all in one record)
- **RC1** — Immutable canonical DAG specification (foundation)
- **PoHT 3.0** — Operational semantics, temporal graph embeddings, runnable demo
- **Profile B** — zk-STARK circuits for verifiable acyclicity + threshold projection
- **Profile C** — Merkle audit logs for institutional oversight & reproducibility

**All extensions are strictly neutral and reference RC1 verbatim.**
